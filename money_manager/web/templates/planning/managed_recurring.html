Design note

Still recurring rules

These pages do not duplicate the money logic. They create and update normal recurring rules, then add contact, variable-amount and manual-check metadata on top.

Workflow

How to use

1. Save the rule here. 2. If you know the next bill value early, write it in Next due amount. 3. When the pending row appears, that value is prefilled. 4. Execute it from this page or Pending.
