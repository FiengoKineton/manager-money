How it works

Add Expense integration

During Add Expense, choose Gift card / Buono sconto balance as discount type and insert the amount you want to use. The receipt saves the source, amount used, balance before and balance after.

Creating a balance inside Add Expense saves it only together with that expense. To save a buono without using it immediately, create it on this page.
